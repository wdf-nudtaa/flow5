\subsection{Common definitions}
The boundary layer thickness $\delta$ is such that $u=0.95\, U_e$.

Non-dimensional wall-normal distance:
$$\eta=\frac{y}{\delta}$$
$y^+$ is a Reynolds number based in the velocity scale $u_\tau$ and length scale $y$ from the wall:
$$y^+ = \frac{y\,u_\tau}{\nu}$$
The wall friction velocity $u_\tau$ gives a turbulence velocity scale because the experimental data  shows that $-\overline{u^{\prime} \, v^\prime}$ is of the same scale of magnitude as $u_\tau^2$ (C\&C section  6.2.1).
$$u_\tau = \sqrt{\frac{\tau_w}{\rho}}$$
Boundary layer Reynolds number:
$$Re_\tau = \frac{u_\tau \, \delta}{\nu}$$
The normalized freestream velocity is $u^+$
$$u^+ = \frac{u}{u_\tau}$$
$$u_e = u_{y=\delta}$$
$$F^\prime(\eta) = u_e^+ - u^+ = \frac{u_e-u}{u_\tau}$$
$$F = \int_0^\eta F^\prime (\xi) d\xi$$
$$ F_1 = F(1)$$
$$\frac{u_e \, \delta^*}{\nu} = F_1 Re_\tau$$
So that
$$ F_1 = \frac{u_e \, \delta^*}{u_\tau \, \delta}$$
